# Remove an Entity

The request below deletes the ***Person*** with ***UserName*** 'vincentcalabrese'.

`DELETE serviceRoot/People('vincentcalabrese')`


Response Payload

```json
HTTP/1.1 204 No Content
OData-Version: 4.0
```
